# Service

## Resize Service

### Event

- default()

  - You can use default() function to get the current size
  - Input => Don't input
  - Output => xs:0 | sm:576px | md:768px | lg:992px | xl:1200px | xxl:1400px

- detectSize()

  - You can use detectSize() function to get the changed size
  - Input => width: number
  - Output => xs:0 | sm:576px | md:768px | lg:992px | xl:1200px | xxl:1400px
